Sketching for Real-time Control of Crowd Simulations

The University of Sheffield (2016 – 2021)

  • Control pedestrian simulations by sketching elements in the environment

  • The system was developed in Unreal Engine

  • The agent-based simulation was developed in CUDA and the FLAME GPU framework

  • Three papers published [1] [2] [3]

Using Human Motion Capture to Control an Anglepoise Lamp

The University of Sheffield (2015 – 2016)

  • Use player movement to control a lamp in a game

  • The objective of the game is to complete an obstacle course

  • The game was developed in Unity

  • The Microsoft Kinect was used to capture the palyer motion

Crowd simulation in Milton Keynes train station area

Transport Catapult Systems, Milton Keynes (2017)

  • Agent-based simulation running in a VR environment

  • Autonomous vehicles added to the simulation

  • VR treadmill integration to interact with the crowd

  • Simulation implemented in Unreal Engine
