PhD

Sketching for Real-time Control of Crowd Simulations (2016 - 2021)

The aim of the work is to control pedestrian simulations in an intuitive manner via sketching. The idea is to control crowd behaviour by sketching multiple elements in the environment while the simulation is running. The user is able to create obstacles to block pedestrian paths; flow lines to force pedestrians to follow a specific path; areas which are used as waypoints; storyboards to define the exact journey of the pedestrians; and simulate events using a timeline. Two navigation methods were explored: navigation grid and navigation mesh. This work produced three publications [1] [2] [3].

  • The system was developed in Unreal Engine 4 mostly in C++. The GUI was created with blueprints

  • The 3D environment model was obtained from OpenStreetMap data, the open source tool OSM2world and Maya

  • The characters were created using AutoDesk's Character Generator

  • A keyframe vertex walking animation was made in 3ds Max

  • The agent-based simulation was developed in CUDA and the FLAME GPU framework

Practical Scenarios

A possible application of the system could be the marshalling of a crowd through a city. An example would be a police force controlling a football crowd moving from a railway station to a football ground.
