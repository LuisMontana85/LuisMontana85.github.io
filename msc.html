Using Human Motion Capture to Control an Anglepoise Lamp (2015 - 2016)

  • The game was developed as the master's dissertation project

  • The aim of the project is to control an object using human motion capture data

  • The goal of the game is to complete an obstacle course

  • The game was developed in Unity

  • The Microsoft Kinect was used to capture the player motion

  • The lamp models were created in Maya

Survival iOS game (2015)

  • The game was developed as part of the Software Development for Mobile Devices masters module

  • Multiple documents were produced: requirements, wireframe design, testing

  • The game consists in the player performing certain actions to survive as long as possible

  • The game was developed in Xcode using Objective-C

  • The animated character was obtained from game2art
